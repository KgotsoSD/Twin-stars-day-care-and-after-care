# Neon Runner - 3D Endless Runner Game

A cyberpunk-themed 3D endless runner game built with React, Three.js, and Vite.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- Three.js (via @react-three/drei)
- shadcn-ui
- Tailwind CSS
- Zustand (state management)

## Building for Production

```sh
npm run build
```

## Deployment

Build the project and deploy the `dist` folder to your preferred hosting service.
